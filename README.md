# Accounting_Dashboard

Supporting dashboard for Fundamentals of Finance & Accounting project

## Usage

### Online

Visit [online hosted version](https://ffa-project.streamlit.app/)

### Local

1. Clone repo
2. Run `streamlit run Accounting_Analysis.py`
